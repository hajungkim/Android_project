# 택시서비스 안드로이드 어플
>  고객이 현주소와 목적지주소를 보내면 기사가 받아서 서비스 하는 어플입니다
## About
- mqtt를 이용하여 두 어플 사이에서 값을 주고 받음
- tmap지도와 gps를 사용하여 주소의 위치 보여줌
- sqlite를 이용해 서비스 이력 저장
## Project stack
- Android studio
- java
- sqlite
## Project period
2017-04 ~ 2017-11
